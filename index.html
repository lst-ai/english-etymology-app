學術引用與來源說明

本學習系統（RootFlow）為供高中學子升學考試及英語字源學自主學習使用，為確保學術嚴謹度、釋義準確度與學習之可靠性，所有內容與底層服務之引用來源聲明如下：

1. 單字字表與難度分級 (Vocabulary & Levels) 預設的 1000 個必背單字與分級主要參考中華民國教育部公布之「高中英文參考詞彙表 (7000字表)」及大考中心 (CEEC) 學術科能力測驗與分科測驗之詞彙範圍。難度劃分邏輯係依據學術界常用之字元長度階層化技術：5 字元以下為「基礎常用 L1-L2」，6 至 8 字元為「學測核心 L3-L4」，9 字元以上為「分科進階 L5-L6」，讓各階段考生循序漸進掌握。
2. 字首字根拆解與詞素釋義 (Etymology & Roots) 本系統之英文單字結構剖析（Prefix / Root / Suffix）及詞源脈絡，係嚴格參考 《Oxford Dictionary of English Etymology》、 《Online Etymology Dictionary (Etymonline)》，以及國內英語教育字源記憶學之主流學術編寫規範。透過邏輯拆解（如 unprecedented = un- (無) + pre- (前) + ced (走) + ent (形容詞字尾) + ed (分詞字尾)），協助學習者用邏輯理解單字，而非死記硬背。
3. 中文翻譯與詞性釋義 (Translations & POS) 單字釋義優先對接教育部國中小暨高中常用單字庫及聯考常考中文翻譯。針對動態匯入、雲端同步或手動輸入之生字，系統提供學術界廣為應用之免費 keyless 翻譯服務 MyMemory API 作為後備翻譯引擎，並經過精細的去冗詞演算法清洗（例如自動去除翻譯結果中贅餘的「一個」、「要」、「去」、「被」等前置詞），確保釋義純淨。
4. 語音發音與美式口音 (Pronunciation) 本網頁所有單字發音及多詞性朗讀，皆採用 HTML5 標準規範之瀏覽器原生發音引擎 Web Speech API (SpeechSynthesis)。預設呼叫標準美式英語（en-US）發音核心，發音速度調整為符合學習者需求之 0.85 倍速，以利精準模仿重音。發音時已透過代碼過濾贅詞，點擊即可聽取最乾淨的原生單字發音。
